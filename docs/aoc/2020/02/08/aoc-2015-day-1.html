The Setup

Problem Description

Santa is trying to deliver presents in a large apartment building, but he can’t find the right floor - the directions he got are a little confusing. He starts on the ground floor (floor 0) and then follows the instructions one character at a time.

An opening parenthesis, (, means he should go up one floor, and a closing parenthesis, ), means he should go down one floor.

The apartment building is very tall, and the basement is very deep; he will never find the top or bottom floors.

Example of Input

  • (()) and ()() both result in floor 0.
  • ((( and (()(()( both result in floor 3.
  • ))((((( also results in floor 3.
  • ()) and ))( both result in floor -1 (the first basement level).
  • ))) and )())()) both result in floor -3.

Question to Answer

To what floor do the instructions take Santa?

Translating the problem into code

The first step when trying to solve any kind of problem with code is determining which data structures in the code we will use to represent various things.

In this case it looks like we will at the very least need to deal with Numbers and Strings.

The floors will be represented as a number:

floor = 0

Moving up a floor can be represented by addition:

floor = floor + 1

Moving down a floor can be represented by subtraction:

floor = floor - 1

And the directions will be represented as strings:

directions = "(())"

Now we need to follow our directions one character at a time so that means we will also need to work with Arrays

directions_array = ["(", "(", ")", ")"]

Describing the solution in English

My floor starts as 0. Given some string of directions I need to:

  1. Turn that string into an array of individual characters
  2. Look at each character in the array one by one
    1. If that character is ( I need to add 1 to my floor
    2. If that character is ) I need to subtract 1 from my floor
  3. Return what floor I am on after I finish looking at all of the characters in the array.

Translating English solution into code

I can use a variable called floor to store the floor that I am on. Initially it is 0

floor = 0

In order to split up a string of directions into an array of individual characters I can use the .split method. See documentation for .split

directions_array = "(())".split("")
#=> ["(", "(", ")", ")"]

I can then use .each on that array to loop through it and adjust my floor based on one character at a time. See documentation for .each

directions_array.each do |character|
  # do something with the character
end

In order to do different things depending on what the character is, I will need to use an if statement. See documentation for if

if character == ")"
  floor = floor + 1
elsif character == "("
  floor = floor - 1
end

Putting it all together

We’re going to define a method that is going to do the work for us, and the input to that method is going to be these directions that we are getting as a string of parentheses.

def follow_directions(directions_string)
  # do stuff here
end

The first thing we are going to put inside our method is going to be the code that handles that first part of our plain english solution

Turn that string into an array of individual characters

def follow_directions(directions_string)
  directions_array = directions_string.split('')
end

The next part of our method is going to initialize the variable that is going to store what floor we are on. So we can call that variable floor and we can set it to 0 to start.

def follow_directions(directions_string)
  directions_array = directions_string.split('')
  floor = 0
end

After that we are going to start working on the second part of our plain english solution.

Look at each character in the array one by one

To do this we are going to use the .each method.

def follow_directions(directions_string)
  directions_array = directions_string.split('')
  floor = 0
  directions_array.each do |direction|
    # do stuff here
  end
end

Inside of this .each method we are going to handle the next two parts of our plain english solution.

If that character is ( I need to add 1 to my floor

If that character is ) I need to subtract 1 from my floor

We are going to accomplish this with the if statement that we talked about earlier.

def follow_directions(directions_string)
  directions_array = directions_string.split('')
  floor = 0
  directions_array.each do |direction|
    if direction == ")"
      floor = floor + 1
    elsif direction == "("
      floor = floor - 1
    end
  end
end

The final part of our method is going to take care of the last part of our plain english solution.

Return what floor I am on after I finish looking at all of the characters in the array.

def follow_directions(directions_string)
  directions_array = directions_string.split('')
  floor = 0
  directions_array.each do |direction|
    if direction == ")"
      floor = floor + 1
    elsif direction == "("
      floor = floor - 1
    end
  end
  floor
end

Testing our method

We can use the examples of inputs and outputs for this problem to test our method.

So we know that the input (()) should produce the output 0

#=> 0

And we know that the input (()(()( should produce the output 3

follow_directions("(()(()(")
#=> 3

When we are running our method and we want to print out the output of the method to the console we can use puts

puts follow_direcitons("(())")

This is fine for manual testing, but manual testing is tedious and error prone, so what I would like to do is write some code that will use all of the examples I have for this problem to automatically test my code for me.

Given an array of examples, where each example is an array where the first item is the input, and the second item is the expected output.

    1. Call the method I want to test using the first part of the example as the input
